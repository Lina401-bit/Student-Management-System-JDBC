# Student Management System

## Project Description

A console-based Student Management System developed using Java, JDBC, and MySQL.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

## Technologies Used

- Core Java
- JDBC
- MySQL
- ArrayList
- Exception Handling
- OOP Concepts

## Database

Run the following SQL script before executing the project.

```sql
CREATE DATABASE studentdb;

USE studentdb;

CREATE TABLE student(
id INT PRIMARY KEY,
name VARCHAR(50),
course VARCHAR(50),
marks DOUBLE
);
```
